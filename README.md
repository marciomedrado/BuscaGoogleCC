# BuscaGoogleCC

Um buscador de imagens especializado em conteúdo com licença **Creative Commons**.

## Objetivo
Facilitar a busca por imagens que podem ser reutilizadas legalmente, utilizando a API de Pesquisa Personalizada do Google.

## Tecnologias
- HTML / CSS / JavaScript
- Google Custom Search API

## Como Usar
1. Clone o repositório.
2. Adicione sua API Key do Google.
3. Abra o `index.html`.
